Setting Game Time

By default the game time is 150 seconds which is 2min 30sec which is the typical time for a game. If however it is different and you want to avoid having the game time change to 150 each time you can change the default game time by doing the following.

1.Change the game time number to the new number under the preparation panel.

2.Click Match from the menu at the top of the scouter

3.Select Change Game Time
