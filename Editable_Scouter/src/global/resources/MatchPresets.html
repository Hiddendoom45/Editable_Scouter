Setting Match Presets

If head scouter gives you a match presets file then in order to load it into the scouter the following must be done.

1.Click Match from the menu at the top of the scouter

2.Select Team Presets

3.Select the given match file, (generally Match.txt), if it is not a txt file and you cannot select it, change file format to All Files.

4.Click open

5.Change the match number to the current match and the team numbers will automatically be loaded. You can manually change the team numbers by clicking on the box with the team number and changing the number if any are wrong. 


Important

Setting the Match Presets must be done every time you launch the scouting program regardless of whether you’ve launched it or not previously.
