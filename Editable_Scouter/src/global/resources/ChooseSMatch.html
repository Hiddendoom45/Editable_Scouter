Choosing a Match

1. To choose a match select from the combo box whether it’s a qualifier match, quarter semi-final match, or finals match. 

2. After that choose the match number to read by typing in the number into the spinner.

3. Press Read Data to read the matches initially(or alternately next/previous,through they’ll read the matches that is 1 less/more than the one you typed in). 

4. Press next/previous to read the previous/next match that was played.


Important

You must set the directory for the match you’re reading otherwise nothing will happen. (This does not apply if the matches are in the same folder as the scouter.jar, but it’s good practice to set the directory).
