Saving a match

When you’ve recorded everything and made all your comments on the team you’re scouting before pressing save. From there you choose the directory where the file is saved, so the folder you see the contents of in the filechooser is where the match file will go. 


The name will be automatically generated and what you enter in the name section will have no effect. If you added some things after you’ve saved the match you can overwrite the file by pressing the save button save again and saving to the same location.


Alternately File>Save does the same thing as the save button in the score panel.


Important

Make sure you save before pressing reset otherwise EVERYTHING is lost.
