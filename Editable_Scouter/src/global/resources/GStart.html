What to Do in Right Before a game starts

First press the setup button on the top left of the game panel(this does some background stuff needed to get some things to work).

After that use the team select combo box to select the team which you’ll be scouting(if you did not fill in the preparation panel properly weird things will happen). If you’re scouting multiple teams change the selection based on which team you’re making a note on. (You can also type a&d respectively into the shortcut box to increment/decrement the selected team).


Then just wait for the game to start and keep the mouse over the start button. (Alternately Ctrl+G)
